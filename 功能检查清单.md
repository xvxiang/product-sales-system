## 🎯 功能演示流程

### 演示场景 1：查看商品列表
1. 访问 http://localhost:8080/index.html
2. 查看商品表格，确认分页正常
3. 查看所有字段显示正确

### 演示场景 2：搜索商品
1. 在搜索框输入"P001"
2. 点击搜索按钮
3. 查看搜索结果

### 演示场景 3：新增商品
1. 点击"新增"按钮
2. 填写商品信息
3. 点击"确定"保存
4. 查看是否添加到列表

### 演示场景 4：编辑商品
1. 点击某行的"编辑"按钮
2. 修改商品信息
3. 点击"确定"保存
4. 查看是否更新成功

### 演示场景 5：删除商品
1. 勾选一条或多条数据
2. 点击"批量删除"按钮
3. 确认删除
4. 查看是否删除成功

### 演示场景 6：查看销售趋势
1. 点击某个商品编号链接
2. 查看弹出的折线图
3. 查看近期一周销售数据

### 演示场景 7：导入数据
1. 准备 Excel 文件
2. 点击"导入"按钮
3. 选择 Excel 文件
4. 点击"开始导入"
5. 查看导入结果

### 演示场景 8：导出数据
1. 勾选几条数据
2. 点击"导出"下拉菜单
3. 选择"导出勾选的"
4. 查看下载的文件
