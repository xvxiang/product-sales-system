## 🚀 快速开始

### 步骤 1：初始化数据库

```bash
# 登录 MySQL
mysql -u root -p

# 执行 SQL 脚本
source D:\code\product-sales-system\src\main\resources\schema.sql
```

### 步骤 2：配置数据库连接

编辑 `application.yml` 文件，修改数据库密码：
```yaml
spring:
  datasource:
    password: 你的数据库密码
```

### 步骤 3：启动应用

方式一：双击运行 `start.bat`

方式二：命令行启动
```bash
mvnw.cmd spring-boot:run
```

### 步骤 4：访问系统

打开浏览器访问：
```
http://localhost:8080/index.html
```

## 📝 使用说明

### 新增商品
1. 点击"新增"按钮
2. 填写商品信息（带*为必填项）
3. 点击"确定"保存

### 编辑商品
1. 点击表格中的"编辑"按钮
2. 修改商品信息
3. 点击"确定"保存

### 删除商品
1. 单选：点击"删除"按钮
2. 多选：勾选多条数据，点击"批量删除"

### 搜索商品
1. 输入商品编号或名称
2. 选择分类（可选）
3. 点击"搜索"按钮

### 查看销售趋势
1. 点击商品编号链接
2. 弹出销售趋势折线图
3. 查看近期一周销售数据

### 导入数据
1. 准备 Excel 文件（参考导入模板.csv）
2. 点击"导入"按钮
3. 选择 Excel 文件
4. 点击"开始导入"

### 导出数据
1. 点击"导出"下拉按钮
2. 选择导出模式：
   - 导出全部：导出所有商品
   - 导出勾选的：导出选中的商品
   - 导出当前页：导出当前页面的商品
